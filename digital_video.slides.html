What is an image?¶

Pixels: Spacial sampling¶

Whereas audio is continous in time, an image is continous in space.

There are an infinite numbers of pixels, the first step to convert our image into a digital image is sampling.

We will split our image in pixels, each pixels will have as value a color.

We are converting our signal from a continous-space to a discrete-space.

def plot_img(plt, size, show_grid=False):
    plt.imshow(img.resize((size, size)))
    plt.set_title("{}x{}".format(size, size))
    plt.xaxis.set_major_formatter(ticker.NullFormatter())
    plt.yaxis.set_major_formatter(ticker.NullFormatter())
    if show_grid:
        plt.yaxis.grid(True)
        plt.xaxis.grid(True)
        plt.xaxis.set_ticks(np.arange(1, size, 1) - 0.5)
        plt.yaxis.set_ticks(np.arange(1, size, 1) - 0.5)
    else:
        plt.axis('off')

        
f, (plt1, plt2, plt3, plt4) = plt.subplots(1, 4, figsize=(20, 20))

def int_to_rgb(color):
    return ((color & 255),
            (color >> 8) & 255,
            (color >> 16) & 255)

def to_rgb(pixel):
    return (
        int(round(256 * pixel[0])), # Red
        int(round(256 * pixel[1])), # Green
        int(round(256 * pixel[2]))  # Blue
    )

img = mpimg.imread('res/trex_small.png')
